34  /**
35    * A class that describes the capabilites (e.g., handling certain types of
36    * attributes, missing values, types of classes, etc.) of a specific classifier.
37    * By default, the classifier is capable of nothing. This ensures that new
38    * features have to be enabled explicitly.
39    * <p/>
40    *
41    * A common code fragment for making use of the capabilities in a classifier
42    * would be this:
43    *
44    * <pre>
45    * public void <b>buildClassifier</b>(Instances instances) throws Exception {
46    * // can the classifier handle the data?
47    * getCapabilities().<b>testWithFail(instances)</b>;
48    * ...
49    * // possible deletion of instances with missing class labels, etc.
50    * </pre>
51    *
52    * For only testing a single attribute, use this:
53    *
54    * <pre>
55    * ...
56    * Attribute att = instances.attribute(0);
57    * getCapabilities().<b>testWithFail(att)</b>;
58    * ...
59    * </pre>
60    *
61    * Or for testing the class attribute (uses the capabilities that are especially
62    * for the class):
63    *
64    * <pre>
65    * ...
66    * Attribute att = instances.classAttribute();
67    * getCapabilities().<b>testWithFail(att, <i>true</i>)</b>;
68    * ...
69    * </pre>
70    *
71    * @author FracPete (fracpete at waikato dot ac dot nz)

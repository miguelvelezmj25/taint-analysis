194       /**
195        * Returns the path of the given {@code file} relative to the
196        * {@link #getBase() base} directory.
197        * <p>
198        * The relative path is evaluated as follows:
199        * <ol>
200        * <li>If {@code base} is the same as {@code file}, "." is returned</li>
201        * <li>If {@code base} is the direct parent directory of {@code file}, the
202        * simple file name is returned</li>
203        * <li>If {@code base} is a non-direct parent directory of {@code file}, the
204        * relative path from {@code base} to {@code file} is returned</li>
205        * <li>If {@code base} is not parent directory of {@code file}, the relative
206        * path is the path from {@code base} to the common ancestor and then to
207        * {@code file}</li>
208        * <li>If {@code base} is not parent directory of {@code file} and the only
209        * common ancestor of the two is the base directory, the absolute path of
210        * {@code file} is returned</li>
211        * </ol>
212        * Examples (same order as above):
213        * <ol>
214        * <li>("/home/john", "/home/john") → "."</li>
215        * <li>("/home/john", "/home/john/notes.txt") → "notes.txt"</li>
216        * <li>("/home/john", "/home/john/documents/important") →
217        * "./documents/important"</li>
218        * <li>("/home/john", "/home/smith/public/holidays.pdf") →
219        * "../smith/public/holidays.pdf"</li>
220        * <li>("/home/john", "/var/tmp/test.out") → "/var/tmp/test.out"</li>
221        * </ol>
222        *

224        * @return the path of {@code file} relative to the {@link #getBase() base}
225        * directory

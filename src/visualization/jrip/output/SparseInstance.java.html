27  /**
28    * Class for storing an instance as a sparse vector. A sparse instance only
29    * requires storage for those attribute values that are non-zero. Since the
30    * objective is to reduce storage requirements for datasets with large numbers
31    * of default values, this also includes nominal attributes -- the first nominal
32    * value (i.e. that which has index 0) will not require explicit storage, so
33    * rearrange your nominal attribute value orderings if necessary. Missing values
34    * will be stored explicitly.
35    *
36    * @author Eibe Frank
37    * @version $Revision: 12472 $
38    */
